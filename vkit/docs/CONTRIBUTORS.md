## How to Add Yourself

Made a contribution? Add your name to this list in your pull request!

1. Add your entry under "Contributors" in alphabetical order
2. Include:
   - Your name
   - GitHub username (link)
   - Brief description of your contributions

Example:
```markdown
**Jane Doe** - [@janedoe](https://github.com/janedoe)
- Added UART agent
- Improved documentation
```
